实用快捷键

发表于2025-07-23|更新于2025-07-24
|浏览量:

这里是一些实用的快捷键,也是我常用的,涉及各个方面,可能有点杂,希望能帮助到你.

计算机系统

Win + R快速指令
Ctrl + Alt + Delete打开资源管理器
Win + E此电脑
Win + D桌面
Win/Alt + Tab切换应用
Win + ⬅/➡/⬆分屏(全屏的时候)
Win + ⬇缩小窗口(此时Win + ⬆变回全屏)
Win + L锁屏

输入法

Ctrl+.切换英文的半角与全角,这样可以做到中文+半角(用于Markdown)

编译器(以PyCharm为例)

Ctrl + F查找
Ctrl + Shift + F全局搜索
Ctrl + C选中当前行
Ctrl + A全选
Ctrl + X剪切
